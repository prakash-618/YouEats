YouEats is a full-stack online food delivery platform built using the MERN stack. It provides users with a smooth browsing, ordering, and checkout experience, along with a dedicated Admin Panel for managing food items in real time.

---

## Features

### User Features
- User Sign-Up / Sign-In with JWT Authentication  
- Browse food items with responsive UI  
- Add to Cart & manage items  
- Place orders securely  
- Stripe payment gateway integration  
- Real-time menu updates from the database  

### Admin Features
- Add new food items  
- Remove/update existing items  
- Instant reflection of changes on the user interface  
- Secure access to admin APIs

### Frontend
- ReactJS
